# beijing fushengji rewrite

北京浮生记重制版

Rewrite [beijing fushengji][url] using Qt.

[url]: https://github.com/chrisguo/beijing_fushengji

Features:
 - all dialogs are reproduced.
 - support sounds.
 - Support Autosave.
 - Support both English and Simplified Chinese.
